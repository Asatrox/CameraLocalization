#### 2. 编写Dataset和Dataloader ✅
> 用`gibson_f`数据集进行训练和测试
> 
> 图片预处理代码参考vggt的load_fn
> 
> dataset参考F3loc中的代码，输出data{img, depth, pose}字典
> 
> 导入图片时，提供了两种方式
> 1) pad:
> 将图片等比例缩放，使得最长边为 518 。如果短边小于 518 ，用 1 进行填充。最终得到 518 × 518 的图像。
> 2) crop:
> 将图像等比例缩放，使得宽为 518 。如果高大于 518， 进行中心裁剪。
